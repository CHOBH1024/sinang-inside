# Sinang-inside (신앙인사이드) ⛪

신앙생활, 관념, 그리고 믿음에 대한 주제로 다양한 차원에서의 밸런스를 측정하고, 분석된 성향 결과 및 통계를 한눈에 보여주는 **신앙생활 밸런스 게임** 웹 사이트입니다.

## 🌟 주요 기능
1. **나만의 신앙 밸런스 진단**: 6가지 깊이 있는 신앙 생활 관련 주제(기도 vs 실천, 문자 vs 상징, 주권 vs 의지 등)에 대해 슬라이더를 조작해 성향을 진단합니다.
2. **결과 요약 및 이미지 공유**: 개별 진단 카드와 가로 막대 그래프 형식의 결과 요약을 확인하고, `html2canvas` 기반의 깔끔한 스크린샷 캡처 기능을 통해 결과를 클립보드에 복사하거나 이미지로 저장할 수 있습니다.
3. **전체 참여자 통계 조회**: 본인 기기뿐만 아니라 구글 스프레드시트 서버와 연동하여 전체 응답자의 평균 및 표준편차 등의 통계 데이터를 실시간 조회할 수 있습니다.
4. **주제 관리자 모드(Admin)**: 톱니바퀴(⚙️) 아이콘을 통해 진단 주제를 추가, 수정, 순서 변경 및 삭제할 수 있으며, 이미지(data URI 형식) 또한 자유롭게 교체하여 커스터마이징 가능합니다.

## 📊 구글 스프레드시트 연동 (결과 취합 설정)

참여자의 진단 결과를 한곳의 구글 스프레드시트에 자동으로 누적 수집하는 설정 방법입니다.

### 1단계 — 구글 스프레드시트 생성
1. [구글 스프레드시트](https://sheets.google.com)로 이동하여 로그인을 진행합니다.
2. **새 스프레드시트**를 생성하고 적당한 제목(예: `신앙인사이드 결과`)을 지정합니다.

### 2단계 — Apps Script 등록
1. 스프레드시트 상단 메뉴에서 **확장 프로그램 → Apps Script**를 선택합니다.
2. 기본으로 작성된 함수를 지우고, 이 저장소의 `google-apps-script.gs` 파일 내용을 모두 복사하여 붙여넣습니다.
3. 💾 저장 (Ctrl + S) 버튼을 누릅니다.

### 3단계 — 웹 앱 배포
1. 편집기 우측 상단 파란색 **배포 → 새 배포**를 누릅니다.
2. 유형 선택 톱니바퀴(⚙️)를 눌러 **웹 앱**을 선택합니다.
3. 다음 옵션을 설정합니다:
   - **설명**: `신앙인사이드 수집기`
   - **다음 사용자로 실행**: `나(본인 계정)`
   - **액세스 권한이 있는 사용자**: **`모든 사용자`** (반드시 '모든 사용자'로 지정해야 다른 접속자들도 전송이 가능합니다)
4. **배포** 버튼을 클릭합니다.
5. 최초 실행 시 나타나는 **권한 승인** 창에서 본인 계정을 선택하고 `고급 → (안전하지 않음) 이동`을 거쳐 **허용**을 누릅니다.
6. 배포가 완료되면 생성된 **웹 앱 URL**(끝부분이 `/exec`로 끝남)을 복사합니다.

### 4단계 — 신앙인사이드 사이트에 등록
1. 배포된 신앙인사이드 웹페이지 우측 하단의 **⚙️(톱니바퀴)** 아이콘을 클릭합니다.
2. 맨 위 **"결과 수집 서버 (선택)"** 입력칸에 복사한 웹 앱 URL을 붙여넣습니다.
3. **저장하기** 버튼을 클릭하면 완료됩니다.
